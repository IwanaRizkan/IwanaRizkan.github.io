// PERAN JETT SEBAGAI DUELIST // Mewakili negara asalnya, Korea Selatan, gaya bertarung Jett yang tangkas dengan banyak pengelakan memungkinkannya mengambil risiko yang tak bisa dilakukan orang lain. Dia mengitari tiap pertempuran, menebas musuh sebelum mereka menyadari apa yang terjadi.

// SKILL JETT //
{ Q } - Updraft // Spesial skill ini merupakan kemampuan dasar Jett yang membuat dirinya melompat dan mendorong tinggi ke udara.

{ E } - Tailwind // Jika gamer menggunakan skill Signature ini pada Jett, maka ia akan melakukan dash atau gerakan cepat ke arah yang dituju, sedangkan ketika ia dalam kondisi diam akan bergerak ke arah depan. Hal ini sangat bermanfaat untuk mengagetkan dan menyerang lawan secara mendadak.

{ C } - Cloudburst // Dengan skill ini, Jett melemparkan bola angin yang dapat membesar ketika mengenai dinding. Bola yang mengembang tersebut berfungsi untuk menghalangi visi penglihatan lawan saat menabrak permukaan tersebut. Skill ini bisa melengkungkan arah dengan cara menahan tombol dan membelokkan arah kursor ke samping kiri atau kanan.

{ X } - Blade Storm //Blade Storm merupakan skill Ultimate milik Jett. Serangan ini berupa melayangkan satu set pisau dan melemparkannya dengan akurat ke arah kepala lawan. Skill ini dapat diisi ulang ketika berhasil mengeliminasi lawan tersebut. Jett Valorant memiliki skill pasif yang dapat melayang dan meluncur di udara ketika turun dari sebuah kotak atau gedung, dengan menekan spasi pada keyboard. Itulah profil dan skill Jet di dalam game Valorant.

// PERAN REYNA SEBAGAI DUELIST // Tertempa di jantung Meksiko, Reyna mendominasi kombat tunggal dan menjadi makin kuat tiap berhasil membunuh. Kepiawaiannya hanya terhambat oleh keahlian mentahnya, membuatnya sangat bergantung pada performa.

// SKILL REYNA //
1. Skill Leer // Skill ini memiliki fungsi yang mirip seperti flash, Reyna dapat membutakan pandangan musuh bila mereka melihat arah ke energi bola berbentuk mata yang ia keluarkan. Berbeda dengan flash milik Breach atau Phoenix yang bekerja secara seketika, skill leer milik Reyna bekerja dengan menampilkan semacam orb yang muncul di hadapan musuh. Karena menampilkan langsung obyeknya, musuh pun bisa menghancurkan orb tersebut. Kamu memiliki waktu 2 detik ketika skill ini dikeluarkan.

2. Skill Devour // Skill yang paling sering Reyna gunakan demi membuatnya menjadi karakter yang tanky. Setiap kamu berhasil membunuh musuh, kamu bisa mengaktifkan skill ini untuk memulihkan health yang berkurang bahkan mengisinya sampai melebihi batas (150 Health). Untuk memulihkan health yang berkurang, jangan ragu untuk mengaktifkan skill ini. Poin dari health tersebut akan berkurang seiring berjalannya waktu. Sehingga tentu ada baiknya kamu selalu untuk aktif berbaku tembak atau melacak musuh ketimbang bertahan saja.

3. Skill Invisible Dismiss // Skill ini akan membuat Reyna Valorant menghilang selama 3 detik ketika mengkonsumsi Soul Orb. Reyna tidak akan bisa ditembak oleh musuh ketika mengaktifkan skill ini. Jika dikombinasikan dengan kemampuan utamanya, Reyna juga tak akan terlihat selama mengaktifkan Empress. Kombinasi ini sangat mengerikan, sebab musuh akan kesulitan untuk memastikan lokasinya dan menjatuhkan Reyna. Sementara sang duelist satu ini tetap bisa menyerang dalam wujud tak terlihat.

4. Skill Ultimate - Empress // Kecepatan menembak Reyna Valorant akan meningkat ketika skill ultimate ini diaktifkan. Agen ini juga bisa melakukan penukaran senjata dan reload di atas kemampuan rata-rata. Reyna akan memasuki dunia yang membuat kecepatannya bertambah secara drastis ketika skill ini aktif. Skill ini akan lebih mematikan jika dikombinasikan dengan skill Dismiss yang dimilikinya. Maka tak heran jika Reyna akan menjadi agent Valorant favorit para pemain duelist

// PERAN NEON SEBAGAI DUELIST // Agen dari Filipina, Neon, melesat maju dengan kecepatan yang mencengangkan, menyemburkan pancaran bioelektrik secepat tubuhnya menghasilkannya. Dia berlari untuk mengejutkan musuh, lalu menyerang mereka lebih cepat dari kilat.

// SKILL NEON //
{ E } - High Gear // Neon memiliki kemampuan untuk melakukan sprint dan slide dengan kemampuan High Gear. Neon bisa berlari dengan cepat memutari map dengan kemampuannya ini. Ketika Neon berlari dia bisa melakukan sebuah slide yang membuatnya meluncur dengan lututnya.

{ Q } - Relay Bolt // Neon memiliki kemampuan untuk melemparkan sebuah bola energi yang bisa memantul pada permukaan yang dikenainya. Bola energi ini nantinya memiliki efek ‘menggegarkan’ (mirip skill Breach dan Astra) dan memiliki 2 hulu ledak. Bola energi ini akan meledak ketika kontak dengan tanah.

{ C } - Fast Lane // Neon bisa mengaktifkan dua tembok api di destinasi yang ia pilih. Kemampuan ini hampir mirip dengan tembok api milik Phoenix namun bedanya tembok ini memiliki dua sisi, berwarna biru, dan tidak bisa dilengkungkan. Kemampuan Fast Lane ini bisa digunakan sebagai langkah perlindungan ketika Neon atau tim masuk ke medan tempur.

{ X } - Overdrive (Ultimate) // Ultimate milik Neon dinamai Overdrive yaitu pancaran listrik yang keluar dari ujung jari Neon. Ketika melakukan ultimate, kecepatan Neon bertambah seperti dia sedang dalam masa High Gear. Kemampuan ini akan reset(berulang) jika Neon mendapatkan sebuah kill(mirip ultimate Jett).

// PERAN CYPHER SEBAGAI SENTINEL // Seorang makelar informasi dari Maroko, Cypher adalah jaringan pengintai satu orang yang mengawasi tiap gerakan musuh. Tak ada rahasia yang aman. Tak ada manuver yang tak terlihat. Cypher selalu mengawasi.

// SKILL CYPHER //
{ Q } - Cyber Cage // Dengan menekan tombol Q pada keyboard dalam bermain Valorant, jika menggunakan hero ini akan mengeluarkan sebuah bola biru yang dapat membuat sebuah ruang berbentuk bulat. Zona atau ruang ini berfungsi untuk menghalangi pandangan lawan dan jika lawan memasukinya maka akan memperlambat Movement Speed.

{ E } - Spycam // Dengan melakukan spesial skill ini, Cypher akan menempatkan kamera pengintai di tempat yang sudah diarahkan. Setelah tertempel, jika ada lawan yang bersembunyi dan terlihat oleh kamera tersebut, tembakkan peluru penanda agen lawan agar lokasi dan pergerakannya bisa menembus dinding.

{ C } - Trapwires // Skill ini merupakan sebuah kawat perangkap yang dapat digunakan pada posisi menyerang dan bertahan. Ketika bertahan akan memberitahu lokasi lawan, jika menyerang akan menghasilkan efek Dazed dan tidak bisa menggunakan hero selama beberapa saat.

{ X } Neutral Theft // Skill Ultimate milik Cypher ini dapat dilakukan ketika lawan yang sudah tereliminasi dengan menembakkan amunisi berwarna biru ke arahnya. Kemudian efek yang didapatkan adalah Cypher dapat mengetahui semua lokasi lawan yang masih hidup di sekitar peta.

// PERAN RAZE SEBAGAI DUELIST // Raze membawa kemeriahan dari Brasil bersama kepribadian besar serta senapan besarnya. Dengan gaya bermain trauma benda tumpul, dia unggul dalam menyapu musuh bertahan dan mengosongkan ruang sempit dengan ledakan besar.

// SKILL RAZE //
Skill 1 – Boom Bot // Boom bot adalah sebuah robot kecil yang dapat digunakan untuk meng-scan suatu lokasi di dalam map.

Skill 2 – Blast Pack // Blast Pack adalah sejenis C4 yang dapat menempel di dinding dan lantai dan akan meledak jika kamu aktifkan atau timernya habis.

Skill 3 – Paint Shells // Pada dasarnya, paint shells adalah sebuah granat yang akan memberi damage saat meledak. Namun bedanya, saat meledak paint shells akan membentuk serpihan baru dan akan meledak kembali, memberi damage yang lebih besar.

Skill Ultimate – Showstopper // Showstopper adalah skill paling mematikan yang Raze punya. Pada dasarnya saat kamu sudah mengoleksi 7 ult points, kamu akan mendapatkan bazooka yang dapat membunuh satu skuad dalam satu tembakan.
